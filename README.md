# TAK Gateway

TAK client connectivity (ATAK/WinTAK) for CNAK. See [CNAK docs](https://github.com/cnak-us/cnak) for full documentation.

## License

This software implements the TAK client protocol (Cursor on Target) and interoperates with [TAK Server](https://github.com/TAK-Product-Center/Server) and TAK clients (ATAK, WinTAK, iTAK).
